## 📖 Sobre o projeto

Este repositório reúne todos os meus estudos da **Maratona Java**, documentando minha evolução na linguagem Java por meio de exercícios, exemplos práticos e pequenos projetos.

O objetivo é consolidar os principais conceitos da linguagem, desde os fundamentos até tópicos avançados de Programação Orientada a Objetos e do ecossistema Java.

---

## 🎯 Objetivos

- Aprender Java do básico ao avançado
- Dominar Programação Orientada a Objetos
- Desenvolver boas práticas de programação
- Criar uma base sólida para Spring Boot
- Construir um portfólio consistente de projetos Java

## 🚀 Como executar

Clone o repositório:

```bash
git clone https://github.com/davimj99/MaratonaJava.git
```

Entre na pasta:

```bash
cd MaratonaJava
```

Abra o projeto no **IntelliJ IDEA** e execute a classe desejada.
